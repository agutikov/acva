# S9.2.2 — FSM `SpeculativeThinking` sub-state

**Phase:** P9.2 (speculative LLM start)  
**Estimate:** 4-5 days  
**Depends on:** S9.2.1 gate policy.

## Context

Adds a concurrent FSM sub-state. `SpeculativeThinking` runs
*alongside* `UserSpeaking` / `Transcribing` — the LLM is
generating against the partial while the user keeps talking.
On `FinalTranscript`, reconcile: prefix matches → keep the
speculative run; differs → cancel and restart.

## Tasks

### Task T9.2.2.1 — FSM extension

Modify `src/dialogue/fsm.{hpp,cpp}`:

```cpp
struct FsmSnapshot {
    State state;
    TurnId active_turn;
    TurnOutcome outcome;
    bool      speculative_in_flight;   // NEW
    TurnId    speculative_turn;        // NEW: 0 when not speculating
    std::string speculative_prefix;    // NEW: for reconcile
};
```

`SpeculativeThinking` is not a top-level state (`state`
remains `UserSpeaking` or `Transcribing`); it's a flag +
shadow turn. Two new transitions:

- `UserSpeaking + GateDecision::Start → speculative_in_flight = true`
- `Transcribing + FinalTranscript → reconcile()`

### Task T9.2.2.2 — Manager wiring

`src/dialogue/manager.cpp`:

- Subscribe to `PartialTranscript`, run `SpeculationGate::evaluate()`.
- On `Start`: assign a fresh `TurnId`, build prompt against
  the stable prefix, fire `LlmStarted` with the speculative
  turn id, route the LLM stream into a holding buffer (don't
  speak yet — sentences accumulate but `TtsBridge` doesn't
  see them).
- On `Replace`: cancel the speculative turn (existing
  cancellation cascade handles drains), repeat `Start`.
- On `Cancel` (or FSM leaves UserSpeaking): cancel; don't
  restart.

### Task T9.2.2.3 — Reconcile

In `Manager::on_final_transcript`:

```cpp
if (speculative_in_flight) {
    if (final_text.starts_with(speculative_prefix)) {
        // Promote the speculative turn — switch active_turn,
        // pop accumulated sentences into TtsBridge,
        // continue generation if final_text extends the prefix.
        promote_speculation(final_text);
    } else {
        // Diverged — cancel speculative, start fresh.
        cancel_speculation();
        start_thinking(final_text);
    }
}
```

Edge case: speculative LLM finished generation before
`FinalTranscript` arrived. The accumulated sentences are
held; reconcile decides whether to play or drop them.

### Task T9.2.2.4 — Tests

`tests/test_speculation_reconcile.cpp` (new) — at least
6 scenarios:

| Scenario | Outcome |
|---|---|
| Final == speculative prefix exactly | Promote, no LLM restart |
| Final extends speculative prefix | Promote, continue generation |
| Final diverges after 3 tokens | Cancel speculative, restart |
| User barge-in mid-speculation | Cancel speculative, no restart |
| Speculative finished early; final matches | Play accumulated sentences |
| Speculative finished early; final diverges | Drop accumulated sentences, restart |
