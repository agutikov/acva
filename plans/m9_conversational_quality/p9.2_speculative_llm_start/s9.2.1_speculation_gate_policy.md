# S9.2.1 — `SpeculationGate` policy

**Phase:** P9.2 (speculative LLM start)  
**Estimate:** 4-5 days  
**Depends on:** P9.1 partials flowing.

## Context

When a `PartialTranscript` arrives, decide whether to
speculatively start an LLM run on its stable prefix. The wrong
gate fires too eagerly (LLM cost on every partial, more
cancellations than completions) or too late (no latency win
over waiting for the final).

## Tasks

### Task T9.2.1.1 — `dialogue::SpeculationGate` interface

`src/dialogue/speculation_gate.{hpp,cpp}` (new). Pure logic;
no I/O. Inputs per call: most-recent `PartialTranscript`,
prior speculation state, current FSM state. Output:
`{decision: {start, hold, replace, cancel}, prompt_text:
string}`.

```cpp
enum class GateDecision : uint8_t {
    Start,    // begin a new speculative LLM run
    Hold,     // keep the current speculation, no change
    Replace,  // cancel current, start a new one with different prefix
    Cancel,   // cancel current, no replacement (e.g., FSM left UserSpeaking)
};
```

### Task T9.2.1.2 — Policy criteria

Initial heuristics (tunable via config):

- **Stability** — `stable_prefix_len ≥ N tokens` (default 4)
  for `M consecutive partials` (default 2).
- **Length floor** — total stable prefix is `≥ K chars`
  (default 12) — too short and the LLM can't predict
  meaningfully.
- **VAD agreement** — last frame's VAD probability is
  decreasing (the user is winding down) OR partial growth
  has slowed (interval between last 2 partials > median
  inter-partial). Both indicate "they're about to stop".
- **No active speculation** — only one speculative slot
  per turn. If one exists with a different prefix and the
  new prefix would change > X chars, `Replace`; else `Hold`.

All thresholds in `cfg.dialogue.speculation.*`. Hot-reloadable.

### Task T9.2.1.3 — Tests

`tests/test_speculation_gate.cpp` — table-driven, ~30 cases.

| Scenario                                          | Expected |
|---|---|
| First partial, < 4 tokens                          | Hold |
| First partial, 6 tokens, stable_prefix_len=6, 1 partial | Hold (need 2 stable) |
| Second partial, same stable_prefix_len, 6 chars      | Hold (length floor) |
| Second partial, same stable_prefix_len, 30 chars     | Start |
| Active speculation, partial extends prefix by 2 tokens | Hold |
| Active speculation, partial diverges after 2 tokens   | Replace |
| FSM leaves UserSpeaking (FinalTranscript)             | Cancel |
