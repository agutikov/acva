# S14.4.1 — Cross-platform packaging

**Phase:** P14.4 (cross-platform packaging)
**Estimate:** 2-3 weeks
**Status:** stage placeholder.

## Tasks

### T14.4.1.1 — macOS .dmg

`packaging/macos/build-dmg.sh`. App bundle layout, code
signing (Apple Developer ID), notarization.

### T14.4.1.2 — Windows .msi or chocolatey

WiX-based `.msi` for traditional install, OR chocolatey
package for opt-in users. Pick one for v1.

### T14.4.1.3 — winget submission

If chocolatey, also submit to winget for the Windows 11
default package manager.

### T14.4.1.4 — Per-platform release notes

Distinct `RELEASE_NOTES_macOS.md` / `_Windows.md` /
`_Linux.md` for platform-specific gotchas.
