# S14.6.3 — Tauri alternative

**Phase:** P14.6 (desktop UX wrappers)
**Estimate:** 3-4 days
**Status:** stage placeholder.
**Shape:** investigation only.

## Context

Electron's lighter cousin: Tauri uses the OS-native webview
(WebKit on macOS, WebView2 on Windows, WebKitGTK on Linux),
ships as a small Rust binary (~10 MB), and exposes a similar
desktop-app shape.

## Tasks

### T14.6.3.1 — Survey

vs Electron:
- ~10 MB vs ~200 MB binary
- ~50 MB vs ~150 MB idle RAM
- relies on OS webview — version variance is the
  catch
- Rust toolchain instead of Node

### T14.6.3.2 — Prototype + size check

Same hello-world that loads the Web UI. Measure binary +
idle.

### T14.6.3.3 — Decision

Closure note. Tauri is often the right answer when an
operator wants a "real app" feel without the bloat — but
it ships a Rust toolchain dep.
