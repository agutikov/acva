# S10.5.1 — Audio + AEC + log/metric diagnostics

**Phase:** P10.5 (audio + diagnostics panels)  
**Estimate:** 2-3 weeks  
**Status:** stage placeholder.

## Likely tasks

### T10.5.1.1 — VAD + RMS scope

Live oscilloscope-style panel: input level (RMS), VAD
probability, endpointer state. Reads from new SSE event
stream emitting frame-summary events at 10 Hz.

### T10.5.1.2 — AEC ERLE history

Time-series of `voice_aec_erle_db` and
`voice_aec_delay_estimate_ms`. When in-process APM is
disabled (default per L7), panel shows "system AEC active"
banner instead.

### T10.5.1.3 — Playback queue + underruns

Queue depth over time, underrun events as markers. Useful
for tuning `cfg.playback.prefill_ms`.

### T10.5.1.4 — Log viewer

Tail `~/.local/share/acva/log/acva-*.log`. Filter by
component + level. Search. Optional: jq-style filter
expressions.

### T10.5.1.5 — Metric explorer

PromQL-lite query against the embedded Prometheus client
(or proxy through if Prometheus is running alongside). For
the "I just want to see this counter" case without opening
Grafana.

### T10.5.1.6 — Debug-bundle export

One-click "package my last 10 min" — zips up logs, current
config, last N turns from SQLite (PII-redacted), `/status`
snapshot, `/metrics`. Useful for filing bugs with reproducible
context.
