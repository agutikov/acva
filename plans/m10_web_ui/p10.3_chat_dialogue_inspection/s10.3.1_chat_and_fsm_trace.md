# S10.3.1 — Chat surface + FSM/turn trace viewer

**Phase:** P10.3 (chat & dialogue inspection)  
**Estimate:** 2-3 weeks  
**Status:** stage placeholder.

## Likely tasks

### T10.3.1.1 — Text chat panel

Mirror `acva --stdin`. Type a prompt, see streamed reply.
Sub-components: input box, message list with role + lang +
sentence-by-sentence streaming, "stop" button (sends
`POST /mute` + cancel).

### T10.3.1.2 — Live FSM viewer

State machine diagram rendered (e.g., via `mermaid` or a
hand-rolled SVG). Active state pulses; transitions animate
on events.

### T10.3.1.3 — Per-turn span viewer

Click a turn, see the per-stage timeline: VAD → STT (with
partial trail) → LLM → TTS → playback. Time origin =
`SpeechStarted`. Reuses M8B OTLP turn span data when
available; falls back to event-bus reconstruction.

### T10.3.1.4 — Sentence playback timeline

For the active speaking turn: show each sentence as a bar,
with first-audio + finished timestamps, ABOVE/BELOW the
prefill threshold. Useful for tuning the streaming-TTS
chunk-arrival jitter.

### T10.3.1.5 — Acceptance

A new operator can open the UI, type a prompt, see the LLM
stream, click into the turn after, understand what happened
end-to-end.
