# S11.4.3 — Quiz / flashcard tool

**Phase:** P11.4 (reference tools)
**Estimate:** 4-5 days
**Status:** stage placeholder.
**Shape:** impl.

## Context

`ideas.md` line 83: "quiz". Promoted from optional task in
s11.4.1 to a dedicated stage.

Use case: operator asks "quiz me on what I told you
yesterday about Python". Tool reads from the M11.P11.3 KG
+ session memory, produces N questions with answers, drives
a turn-by-turn quiz session.

## Tasks

### T11.4.3.1 — Question-generation prompt

LLM-side: a sub-prompt that asks for N {question, answer,
hint} triples given a topic + memory context.

### T11.4.3.2 — Quiz state

Tool maintains a quiz session: which question is current,
what the user said, scoring. Persists in SQLite as a
side-table; survives warm restart (M8A) by keying on
session_id.

### T11.4.3.3 — Voice UX

Each question is spoken; user answers verbally; LLM judges
and gives feedback. End of quiz, brief summary.

### T11.4.3.4 — Acceptance

10-question quiz on a known topic runs end-to-end with
zero stuck-state events; final score is computed and
spoken.
