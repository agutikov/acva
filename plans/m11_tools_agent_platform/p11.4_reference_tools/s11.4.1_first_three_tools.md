# S11.4.1 — Reference tools: config, web search, KG query

**Phase:** P11.4 (reference tools)  
**Estimate:** 2-3 weeks  
**Status:** stage placeholder.

## Likely tasks

### T11.4.1.1 — Config-modify-and-restart tool

User says "make yourself faster" → tool reads current
config, proposes a hot-field diff, applies via
`POST /reload`. Reuses M8A reload. The first dogfood-
worthy tool.

### T11.4.1.2 — Web search tool

Wrap a lightweight search engine (DuckDuckGo HTML scrape
or Searxng if the operator has one running locally).
Subprocess sandbox (Q6 baseline). No API keys for the
default install.

### T11.4.1.3 — Knowledge-graph query tool

Direct retrieval against the M11.P11.3 store. The LLM
uses it for "what did I tell you yesterday about X".

### T11.4.1.4 — Optional ones (time-permitting)

- Diagram generator (mermaid → PNG)
- Quiz / flash-card generator
- Markdown / text-file scratchpad

These get pulled in if the phase has runway. Otherwise
they live in the ideas backlog under
[`open_questions.md`](../../open_questions.md) Q4-style
parking.

### T11.4.1.5 — Tool authoring template

`tools/example-tool/` — a template repo new operators
copy to start their own tool. Reference for the M11.P11.5
docs.
