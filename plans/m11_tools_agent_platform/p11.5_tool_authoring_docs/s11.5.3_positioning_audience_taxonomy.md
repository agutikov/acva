### T11.5.3.2 — Audience matrix

Five audiences from the ideas list (+ "what else?"):
1. End user (Voice UX only)
2. Advanced user (Web UI editor)
3. Context engineer (memory / sessions / history)
4. Agent engineer (tools / KG / integrations)
5. Runtime engineer (FSM / pipeline / modules)

For each: required surfaces in acva, required docs, current
gaps, milestone that closes them. The matrix lives in
`docs/positioning.md`.

serve audience 4-5? File any gaps as new tasks in the
