## Tasks

Insert N=10k facts; query top-k similarity with k ∈ {5,
20}. Capture latency P50/P95, RAM, disk usage for:
- sqlite-vec (baseline)
- qdrant (sidecar)
- chroma (in-process Python; eliminate if it requires
  Python at runtime)
- lancedb (file-format; in-process)

### T11.3.3.2 — Failure modes

Each backend's recovery from crash, schema migration,
backup/restore. acva's pillar #6 (crash recovery) is
SQLite-grade today; alternatives must match.

### T11.3.3.3 — Recommendation

  facts."
- "lancedb is a free upgrade — same in-process model,
  better at scale, swap when fact count > 50k."

  hitting that threshold.
